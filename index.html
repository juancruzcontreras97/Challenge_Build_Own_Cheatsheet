HTML Form Reference

Form Tags

Tag Name Description
<form> Form Used to create an HTML form for user input
<input> Input Specifies an input field where the user can enter data
<textarea> Text Area Defines a multi-line text input control
<button> Button Defines a clickable button
<select> Select Used to create a drop-down list
<option> Option Defines an option in a select list
<optgroup> Option Group Used to group related options in a drop-down list
<fieldset> Field Set Used to group related elements in a form
<label> Label Defines a label for a certain element
<output> Output Used to represent the result of a calculation (like one performed by a script)
